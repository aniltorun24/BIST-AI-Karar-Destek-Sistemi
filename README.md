# Project Overview

This repository contains the BIST-AI-Karar-Destek-Sistemi project, an AI-based decision support system designed for users analyzing stock market data. The goal is to provide insights and recommendations based on historical data and predictive models.

## Setup Instructions

1. **Clone the Repository**  
   Use the following command to clone the repository to your local machine:
   ```bash
   git clone https://github.com/aniltorun24/BIST-AI-Karar-Destek-Sistemi.git
   ```

2. **Navigate to the Project Directory**  
   Change to the directory of the cloned repository:
   ```bash
   cd BIST-AI-Karar-Destek-Sistemi
   ```

3. **Install Dependencies**  
   Make sure you have Python installed, then install the required dependencies using:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application**  
   Start the application by running:
   ```bash
   python main.py
   ```

For further documentation and usage details, refer to the wiki or additional files in the repository.
